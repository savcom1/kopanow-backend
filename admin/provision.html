📋 How to provision a phone (QR — no USB)

  1. Factory reset the borrower's phone
    Settings → General Management → Reset → Factory Data Reset
  2. On the Welcome / Hi there screen, tap the screen 6 times quickly
    The phone will open the QR code scanner
  3. Scan the QR code above. The phone will:
    • Connect to the internet
    • Download and install the Kopanow app automatically
    • Set Kopanow as Device Owner (deep lock enabled)
  4. Open the Kopanow app and register the borrower (name, ID, loan ID)
  5. Hand phone to borrower. All future locking is remote via this admin panel.

🔌 Alternative: USB Setup USB required

  1. Factory reset the phone and complete Android setup (skip Google account)
  2. Enable USB Debugging: Settings → Developer Options → USB Debugging
  3. Plug phone into your laptop via USB
  4. In PowerShell from the project folder, run:
    .\setup_device_owner.ps1
  5. Open Kopanow app, register borrower, hand to borrower
